Angular TabList component

Simple "multi-level data in the table" component for AngularJS.

Code on GitHub

Why

HTML does not have component which is a tabular but in the same time can have multiple nested levels.

What

"Angular TabList" is the AngularJS directive to present multi-level lists in the tabular form.

Features

  • Behaves like an HTML table
  • Unlimited level of children
  • Sorting of the elements
  • Easy to extend functionality (e.g. Drag & Drop or in-line editing)

Installation:

Install with Bower

Angular TabList is the Bower package, so if you're using Bower then simply add "angular-tablist" as a dependency.
  1. Install Angular TabList as a dependency with Bower:

    bower install angular-tablist -S

  2. Add Angular TabList script to your HTML file: <script type="text/javascript" src="bower_components/angular-tablist/dist/angular-tablist.js"></script>
  3. Add Angular TabList CSS style file to your HTML file: <link rel="stylesheet" type="text/css" href="bower_components/angular-tablist/dist/style/angular-tablist.css">
  4. Add "angular-tablist" as a requirements to your Angular module:

    angular.module("myProjectModuleName", ['angular-tablist']);

Install manually

  1. Go to Angular TabList download page.
  2. Download latest version (currently 1.1.0).
  3. Extract archive into your project.
  4. Add Angular TabList JS script to your HTML file: <script type="text/javascript" src="angular-tablist/dist/angular-tablist.js"></script>
  5. Add Angular TabList CSS style file to your HTML file: <link rel="stylesheet" type="text/css" href="angular-tablist/dist/style/angular-tablist.css">
  6. Add "angular-tablist" as a requirements to your Angular module:

    angular.module("myProjectModuleName", ['angular-tablist']);
